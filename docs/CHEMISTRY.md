(a_i = gamma_i * m_i), gamma по Davies. Скейлы: CaCO3, CaSO4, BaSO4, SrSO4, Halite.

приближение с clamp. Пилот: PhreeqPy / Reaktoro-Pitzer (см. LITERATURE.md).
